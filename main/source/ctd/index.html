Welcome to Symbian's Collaborative Test Database (CTD)

The CTD allows the Symbian community to cooperate on running tests on specific versions of the Symbian Platform, and to contribute the test results back so everyone can see what has been tested and what has not yet been tested. You can also check which test cases have "passed" or "failed" status on a release, and add new test cases.

Most common actions:

  •      Test Case Specification
  •      Test Results
To find out more about CTD and how to use it, see our CTD User Guide.

Note* To raise bugs against the Symbian Foundation website (content, functionality or access) please go to webbugs.
